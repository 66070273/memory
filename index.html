📍 MEMORY 01 — ครั้งแรกทื่ได้เจอเธอ

จำครั้งแรกที่เราเจอกันได้ไหม? ตอนนั้นเรากลัวเธอจะไม่ชอบเรามากและตอนนั้นเป็นครั้งแรกที่นัดเจอคนเราก็แอบกลัวนะ แต่พอเราถึงเราก็โทรมาเธอ ถามว่าเธออยู่ตรงไหน เธอก็บอกให้เราเดินมา เราก็เดินไปเจอผู้หญิงน่ารักคนนึงใส่แมสแอบอยู่หลังเสาในรูป พอมาเจอจิงๆเธอแม่งโครตน่ารัก คนไรน่ารักขนาดนี้ เรารู้สึกโชคดีมากเลย

แตะที่รูปเพื่อดูรูปถัดไป ✨
🍲 MEMORY 02 — ร้านแรกที่เราไปเดตกัน

ร้านนี้เป็นร้านแรกที่เรามาเดตกัน เรายังจำเมนูที่เรากินด้วยกันได้อยู่เลย เป็นไก่ทอดกับบิงซูเมล่อน เราจำได้ว่าเราขอป้อนเธอด้วยเป็นไก่ทอดแต่กลัวเธอร้อนเราเลยเป่าให้ ตอนนั้นรู้สึกอยากให้เวลาเดินไปช้าๆจัง ตอนเรามองเธอกินเห็นแล้วโครตน่ารัก ตอนเธอเห็นเรามอง เธอก็เขินเอามือปิดหน้า กดที่รูปไปดูเมนูที่เรากินด้วยกันได้นะ

📍 MEMORY 03 — ป้ายรถเมล์ที่เรามาส่งเธอทุกครั้ง

เราไม่ชอบป้ายรถเมล์นี้เลย เพราะมันทำให้เราต้องแยกจากกัน เราอยากอยู่กับเธอนาน ไม่อยากให้เธอกลับเลย ทุกครั้งที่เราส่งเธอขึ้นรถเอาจิงเราก็แอบเศร้าตลอดเลยนะ ตอนนั้นเราเลยพยายามมาหาเธอมาเจอเธอตลอดเลย เราอยากอยู่กับเธอนานๆ อยู่กับเธอตลอดเลย

📍 MEMORY 04 — ปากซอยบ้านของเธอ

ครั้งแรกที่เราได้ชึ้นรถเมล์แล้วไปส่งเธอที่หน้าปากซอยบ้านเธอ ตอนนั้นเราอยากไปส่งเธอแถวๆบ้านเธอเลย แต่เธอกลัวพ่อเห็นเราเลยไม่ได้ไปส่งเธอเลย

📍 MEMORY 05 — Airlink หัวหมาก

สถานที่นี้เป็นอีกนึงสถานที่เราสองคนมาบ่อยเวลาเราไปเที่ยวกันเมืองกันเราต้องมาขึ้นไฟฟ้าที่นี้ เป็นสถานที่เป็นความทรงจำเราสองคนที่เยอะเหมือนกัน เราจำได้ว่าตอนนั้นเราอยากจะอุ้มเธอ เพราะไม่อยากให้เธอเดินเหนื่อย เราเลยขอเธออุ้ม แต่เธอก็ไม่ยอมเพราะเขินคน แต่สุดท้ายเราได้อุ้มเธอในท่าเจ้าหญิง

📍 MEMORY 06 — หนังเรื่องแรก

ไม่รู้เธอจะจำได้ไหมว่าเรื่องแรกที่เราดูด้วยกันคือเรื่องไร แต่เรายังจำได้เสมอเป็นหนังเรื่องแรกที่เราได้ดูด้วยกันเป็นความทรงจำที่ดีมาก ถึงเนื้อหาในเรื่องหนังเรื่องนี้มันจะเจ็บปวด แต่เราก็ย้อนกลับไปดูตลอดเลยเพราะมันทำให้นึกถึงเธอตอนดูด้วยกัน

📍 MEMORY 08 — คุ๊กตาตัวแรก

พอเราไปดูหนังกัน แล้วไปกิน pepperlunch กันเสร็จเราก็ไปเล่นตู้คีบตุ๊กตากัน แล้วตัวนี้ก็เป็นตุ๊กตาตัวแรกในชีวิตเราที่คีบได้และเป็นตุ๊กตาตัวแรกที่ให้เธอ ตอนนี้ไม่รู้ว่าน้องยังอยู่มั้ยหรือเธอทิ้งไปแล้ว วันนั้นเป็นวันที่มีความสุขมากจนวันนี้่เรายังจำความรู้สึึกนั้นได้อยู่

📍 MEMORY 07 — PeperLunch ครั้งแรก

ร้านนี้เป็นร้านที่เรามากินกันหลังจบดูหนังเรื่องแรกกัน ตอนนั้นเราเดินหากันว่าจะกินอะไร จนมาจบที่ pepperlunch ซึ่งตอนนั้นเราไม่เคยกินร้านนี้เลย เธอก็ไม่เคย แล้วเราก็สั่งเมนูแบบในรูปเลย เธอกินข้าวเปปเปอร์หมู ส่วนเรากินสปาเก็ตตี๋ เรายังจำได้อยู่ตอนนั้นเราไม่รู้ว่าแต่ละซอสมันปรุงยังไงเราก็เลยใส่ไปทั้งคู่ใส่จนมันเค็มเลย55554 เรารู้สึกผิดมากตอนนั้นทำให้เธอกินข้าวไม่อร่อยเลย แต่ตอนนั้นมันก็เป็นสีสันในชีวิตคู่ของเรานะ เธอถ่ายเราลงด้วยตอนเป็นเชฟกระทะเค็ม

📍 MEMORY 09 — donkki

ร้าน donkki เป็นร้านที่เราไปเดินดูของเล่นกันบ่อยมาก เดินดูนู้นดูนี้แต่มันก็สนุกมากเลยนะ แล้วเราชอบไปเดินตรงโซนที่ขายพวกชุดแล้วเอามาแกล้งเธอ เพราะตอนนั้นเราชอบรีแอคชั่นของเธอ มันดูน่ารักดีเราเลยแกล้งเธอบ่อย ถึงตอนนี้มันจะปิดไปแล้วก็ตาม เราอยาไปเดินเล่นกับเธอแบบนั้นอีกจัง

📍 MEMORY 10 — KariKori

ไม่แน่ใจว่าเธอยังจะจำร้านนีี้ได้อยู่ เป็นร้านที่เรากินกันหลายรอบอยู่ ไม่รู้ว่าเธอยังจำได้มั้ย มีวันนึงที่เราซื้อมาแล้วทำมันหกทำให้เธออดกิน เรารู้สึกผิดและเห็นเธอเศร้า เราเลยไปซื้ออันใหม่ให้เธอ ยังจำรอยยิ้มของเด็กคนนั้นได้อยู่เลย อยากให้ร้านกลับมาเปิดอีกจัง

📍 MEMORY 11 — สวนลุม

เรารู้สึกชอบที่ได้ไปเวลาสวนกับเธอ แบบมันได้ใช้เวลาเงียบๆบรรยากาศดีๆกับเธอไปนาน เหมือนได้พักผ่อนหย่อยใจเลยอะ เราได้ไปเปิดในดงดอกไม้ที่สวยๆมันเหมาะกับเธอมากเลยนะ เราได้มาปั่นเป็ดกันด้วย มันแอบกลัวนะตอนอยู่บนเรือ เพราะแบบว่าเราตัวหนักกว่าเธอแล้วมันเอียงมาทางเรา ตอนนั้นเราคิดต่างๆนาๆเลย แต่พอผ่านไปสักพักมันก็สนุกและเป็นความทรงจำดีๆเลย

📍 MEMORY 12 — เสื้อตัวแรก

นี้เป็นเสื้อตัวแรกที่เราได้ให้เธอ ถึงมันจะไม่ได้สวยเหมือนเสื้ออื่น แต่อยากให้เธอจากใจจิง ตอนที่เธอใส่แล้วถ่ายส่งมามันทำให้ใจเราละลายเลย มันน่ารักสุดๆ แต่ไม่รู้เธอยังเก็บมันไว้อยู่ไหม

📍 MEMORY 13 — วันเกิด

รูปนี้เป็นรูปที่เป็นวันเกิดอายุ 18 ปีของเธอพอดี ตอนนั้นเราไม่ได้มีของขวัญอะไรให้เธอเลย เราทำได้แต่พาเธอมากินข้าว แต่พวกเราก็มีความสุขกันมากจิงๆนะ

📍 MEMORY 14 — ทำงาน

รูปนี้คือตรงร้านที่เคยทำงาน part time ถึงร้านปิดไปแล้วแต่เรายังจำได้เสมอ อาหมวยชุดแดงที่ียืนหน้าร้านคอยต้อนรับลูกค้า เป็นอาหมวยที่เก่งมากเลยที่ผ่านมาได้ ต้องรับมือกับหลายๆอย่างทั้งลูกค้า ทั้งทีร้านเป็นเด็ก 18 ที่เก่งมากๆเราภูมิใจในตัวเธอเสมอนะ แล้วก็ทุกครั้งที่เราไปthe mall เราก็จะแอบดูว่าเธอทำเป็นไง เหนื่อยมั้ยเป็นอีกเรื่องที่เทออาจไม่รู้ พอเราไม่เห็นเธอหน้าร้านเราก็จะทำเป็นเดินผ่านร้านไปเพื่ออยากรู้เธออยู่รึเปล่า

🤍 MEMORY 015 — ครั้งสุดท้าย

นี้คือครั้งสุดท้ายที่เราได้เจอเธอ ถ้าตอนนั้นรู้ว่ามันจะเป็นครั้งสุดท้ายเราจะขอกอดเธอให้นานที่สุดเท่าที่เราจะทำได้ ซึ่งตอนนี้เราพอรู้แล้วว่าเราไม่มีโอกาสนั้นแล้ว สุดท้ายแล้ว...นี้เราอยาก ขอบคุณเธอจริง ๆ ที่ครั้งหนึ่งเธอเคยเข้ามาเป็นเรื่องราวดี ๆ ในชีวิตเรานะ ได้เข้ามาทำให้เราได้เข้าใจกับความรักทำให้เราเติบโต มันเป็นความทรงจำที่ดีที่สุดในชีวิตเราเลยและเราจะไม่ลืมมันแน่นอน แม้สมุดเล่มนี้จะไม่มีหน้าถัดไปให้เขียนต่อแล้ว แต่ทุกหน้าที่เคยเขียนร่วมกันมา จะเป็นความทรงจำที่เราจดจำไว้ด้วยรอยยิ้มและเปิดอ่านดูในใจเสมอไปนะ เราอยากให้เธอรู้ว่าถ้าวันไหนที่เธอเจอกับคนที่โหดร้ายคนที่ทำให้เธอเสียใจหันมาเธอจะเจอเรายืนรอเธอเสมอนะ โชคดีนะ :)
